# Tag

## Tag 的样式

Html 文档由元素组成，元素的语法如下：

> 元素以**开始标签**起始，以**结束标签**终止
> 元素的内容是两个标签之间的内容
> 某些 Html 元素具有空内容(empty content)
> 大多数元素可以拥有他们自己的属性

---

## Tag 的使用

此实例也说明了一个问题，\<h2>可以单独使用，作为二级标题

```HTML
<!DOCTYPE>
<body>
    <p>
        <h1>
            一级标题
        </h1>
        这是一个段落
    </p>

    <p>
        <h2>
            二级标题
        </h2>
        这是一个新段落
        <a href = "https://www.61.com" >这是一个测试链接</a>
        <!-- 请注意：在<a></a>元素中，定义的href若是导向一个新页面，应该以网络协议+网址的形式写 -->
    </p>
</body>
```
